参考

※開発を始める方法は色々あるので一例です。

スケルトンをインストール：

npm install electron-prebuilt -g

配布用パッケージの作成：

electron-packager ./ SampleApp --platform=all --arch=all --version=0.34.3

プラットフォームやバージョンの指定については、 https://github.com/maxogden/electron-packager を参照。プラットフォームが多いと時間はかかります。
